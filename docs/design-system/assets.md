# Visual Assets

The current UI uses Chakra-rendered industrial panels so layout can stabilize
before committing to brand assets.

## Logo

Keep the current `NX` mark as a temporary system mark. Generate or commission a
proper Nexora Systems logo after the company legend and UI direction are
accepted.

Logo requirements:

- works as a compact header mark and full wordmark;
- feels industrial/technical, not consumer SaaS;
- supports light and dark surfaces;
- has accessible alt text;
- can be represented as SVG in the repo after approval.

## Product Imagery

Generated product imagery is useful after page density and hierarchy are
accepted. The first asset set should cover:

- rugged edge gateway;
- wireless vibration sensor;
- monitoring dashboard/SaaS product;
- datasheet/manual/certificate thumbnails;
- release diff or revision proof visual.

Generated images should not replace source-data proof. They support the catalog
experience; Revisium evidence still comes from project/table/row links, diffs,
requests, and responses.
